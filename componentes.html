Componentes del Computador

Inicio Historia Tipos Componentes

En el contexto de la evolución tecnológica, el computador es una de las herramientas más importantes. Está compuesto por diversos elementos que permiten ejecutar tareas, procesar información y conectarse con otros dispositivos.

A continuación, se presentan los principales componentes que forman parte de un computador moderno:

Componentes principales:

  • Procesador (CPU): el “cerebro” del computador, encargado de ejecutar instrucciones.
  • Memoria RAM: almacena datos temporales mientras se realizan operaciones.
  • Almacenamiento: disco duro (HDD) o unidad de estado sólido (SSD) para guardar información.
  • Placa base: conecta todos los componentes entre sí.
  • Fuente de poder: convierte la energía eléctrica para alimentar el sistema.
  • Periféricos: dispositivos externos como el teclado, ratón, monitor e impresora.
